# 66. 加一

TIP

相关标签： 数组

# 题目

给定一个由整数组成的非空数组所表示的非负整数，在该数的基础上加一。

最高位数字存放在数组的首位， 数组中每个元素只存储单个数字。

你可以假设除了整数 0 之外，这个整数不会以零开头。

示例 1:

输入: [1,2,3]
输出: [1,2,4]
解释: 输入数组表示数字 123。

示例 2:

输入: [4,3,2,1]
输出: [4,3,2,2]
解释: 输入数组表示数字 4321。

# 解题思路

# 解法

1.将数组转为字符串

2.字符串转为数字类型

3.使用数字类型进行计算

4.结果再转为字符串

5.字符串转为数组

使用上面的方法普通的测试用例可以通过，但是对于数组长度过长，大于了Number取值范围的就会报错。所以可以采用ES10新出的BigInt类型，具体的使用方法可以看文档。

# 代码

/**
 * @param {number[]} digits
 * @return {number[]}
 */
const plusOne = (digits) => {
  const num = BigInt(digits.join(''));
  const string = String(num + 1n);
  const array = string.split('');

  return array.map(str => Number(str));
};
