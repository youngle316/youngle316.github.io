# JS继承

继承是面向对象中三个特性之一。子类可以继承父类的属性和方法。由于JS与其他的语言不同，那么问题就来了，在JS中类怎么声明，继承的方式又是什么

# 类的声明

# ES5使用构造函数

在ES5中，通常使用构造函数声明一个类

function Animals () {
  this.name = 'dog'
}

# ES6使用class

在ES6中新增了class这个语法糖来声明类，本质上还是与ES5中构造函数相似的

class Fruit {
  constructor (name) {
    this.name = name
  }
}

# 类的实例化

不论是ES5还是ES6都使用 new 关键字来实例化

let dog = new Animals()
let apple = new Fruit('apple')
