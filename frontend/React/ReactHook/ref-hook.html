# React-Hook 之 Ref Hook

Refs提供了一种方式，可以用来允许访问DOM节点，或者在Render中创建的React元素。

# 使用Ref Hook

# createRef

在之前的Class组件或无状态组件中都可以使用createRef()来创建Refs，并通过ref属性添加到React元素上。

下面使用一个获取焦点的例子来演示一下使用方式

  • 为DOM元素添加ref
// class组件
import React, {Component} from 'react';

class InputFocusClass extends Component {
  constructor(props) {
    super(props);
    this.inputElement = React.createRef()
  }
  handleButtonClick = () => {
    this.inputElement.current.focus()
  }

  render() {
    return (
      <div>
        <input ref={this.inputElement}/>
        <button onClick={this.handleButtonClick}>focus</button>
      </div>
    );
  }
}

export default InputFocusClass;

// 函数型组件
import React from 'react';

function InputFocusHook(props) {
  const inputElement = React.createRef()
  const handleButtonClick = () => {
    inputElement.current.focus()
  }
  return (
    <div>
      <input ref={inputElement}/>
      <button onClick={handleButtonClick}>focus</button>
    </div>
  );
}

export default InputFocusHook;

不论是class组件还是函数型组件都可以在其内部使用createRef()来生成Refs。

  • 为组件添加ref
  1. 为class组件添加ref
// InputFocusHook
import React from 'react';
import InputFocusClass from "./InputFocusClass";

function InputFocusHook(props) {
  const inputElementHook = React.createRef()
  const handleButtonClick = () => {
    inputElementHook.current.handleButtonClick()
  }
  return (
    <div>
      <InputFocusClass ref={inputElementHook}/>
      <button onClick={handleButtonClick}>focus</button>
    </div>
  );
}

export default InputFocusHook;


// InputFocusClass
import React, {Component} from 'react';

class InputFocusClass extends Component {

    super(props);
    this.inputElement = React.createRef()
  }

  1. 为函数型组件添加ref

控制台会报这个警告

Warning: Function components cannot be given refs. Attempts to access this ref will fail. Did you mean to use React.forwardRef()?

需要注意的是：不能为函数型组件添加ref，因为函数型组件没有实例。

如果要在函数型组件中使用ref，你可以使用 forwardRef

# useRef

const refContainer = useRef(initialValue);

useRef 返回一个可变的 ref 对象，其 .current 属性被初始化为传入的参数（initialValue）。返回的 ref 对象在组件的整个生命周期内保持不变。

一个常见的用例便是命令式地访问子组件：

import React, {useRef} from 'react';

function InputFocusHook(props) {
  const inputElement = useRef(null)
  const handleInputFocus = () => {
    inputElement.current.focus();
  }
  return (
    <div>
      <input ref={inputElement}/>
      <button onClick={handleInputFocus}>Add</button>
    </div>
  );
}

export default InputFocusHook;
