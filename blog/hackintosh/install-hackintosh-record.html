# 黑苹果安装过程中的一些简单记录

💻从去年就开始折腾黑苹果，想用来做开发。一开始是使用5年前的笔记本(hd4000)来安装黑苹果，成功了。完不完美先不说，就那1366*768的分辨率我看了一下午就有一种眩晕的感觉，想吐。就在夏天的时候组了一个台式机来装黑苹果用于平时的开发和学习。这篇文章就记录一下安装的过程。(注：这不是一篇装黑苹果的教程，只是一个安装过程的记录📝。用于以后重新安装时的笔记吧！) 我安装黑苹果所用的工具和知识，全部来自网上的大神的分享(大神的真滴多)

# 工具

# 从win10做启动器

  • 启动器使用Etcher
  • 安装需要的镜像可以从黑果小兵的博客中下载，或者在远景论坛中也会有分享。它们制作的镜像会直接带有最新的Clover和常用的config，Kext。
  • 8G以上的U盘

# 替换EFI

推荐用上面镜像自带的config安装好，能进入系统后再换EFI。

替换EFI的工具

  • windows下使用DiskGenius
  • mac下使用Clover Configurator，网上有很多，可以自己下载

EFI很重要，推荐先在论坛找别人配好的。如Tonymacx86中的Golden Builds和User Builds。参考别人的电脑配置自己配一台主机。方便省事。
